mantou_mobile
=============

##使用：
1. 使用前准备：npm install && bower install，安装<a href="http://www.imagemagick.org/script/binary-releases.php" target="_blank">imageMagick</a>
2. 起数据库：mongod -dbpath d:/mongodb/mantou_mobile
3. 起服务：gulp

##构建工具：
* gulp

##前端：
* angular
* ionic
* sass

##后端：
* nodejs
* express
* mongodb
* mongoose
* imageMagick
